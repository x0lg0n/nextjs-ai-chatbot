## Troubleshooting

Before opening an issue, please check:

1. Have you followed the [Quick Start Guide](./docs/01-quick-start.md)?
2. Have you set up all required environment variables correctly?
3. Are all dependencies installed properly (`pnpm install`)?
4. Have you checked the [existing issues](https://github.com/vercel/ai-chatbot/issues) for similar problems?

## How to Ask for Help

When asking for help, please provide:

- A clear description of the problem
- Steps to reproduce the issue
- Information about your environment (OS, Node.js version, etc.)
- Any relevant error messages
- Screenshots if applicable
