## Becoming a Maintainer

The Next.js AI Chatbot is maintained by the Vercel team. We generally do not accept external maintainers, but we greatly appreciate contributions from the community.

If you're interested in contributing:
1. Review our [Contributing Guide](./CONTRIBUTING.md)
2. Start by submitting pull requests for small changes
3. Participate in discussions on GitHub issues
4. Help review and test pull requests from other contributors

## Maintainer Responsibilities

Maintainers are responsible for:
- Reviewing and merging pull requests
- Responding to issues and discussions
- Ensuring code quality and consistency
- Maintaining documentation
- Releasing new versions
- Fostering a welcoming and inclusive community

## Contact

For questions about the project or to reach the maintainers, please:
1. Open an issue on GitHub
2. Start a discussion in GitHub Discussions
3. Contact Vercel support for urgent matters
