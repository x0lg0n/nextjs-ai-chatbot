## Overview

The Next.js AI Chatbot is an open-source template built with modern web technologies that provides a foundation for creating AI-powered chat applications. It features:

- Next.js 14 with App Router
- AI SDK integration for multiple model providers
- Blocks system for canvas-like interaction
- Authentication with NextAuth.js
- Database integration with Vercel Postgres
- File storage with Vercel Blob
- shadcn/ui components with Tailwind CSS styling
