# Contributing to Next.js AI Chatbot

Thank you for your interest in contributing to the Next.js AI Chatbot! We welcome contributions from the community and are excited to see what you'll build with this template.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/your-username/ai-chatbot.git`
3. Create a branch for your contribution: `git checkout -b feature/your-feature-name`
4. Make your changes
5. Commit your changes: `git commit -am "Add your feature"`
6. Push to your fork: `git push origin feature/your-feature-name`
7. Create a pull request

## Development Setup

1. Install dependencies: `pnpm install`
2. Set up environment variables (see `.env.example`)
3. Run the development server: `pnpm dev`

## Code Guidelines

- Follow the existing code style
- Write clear, concise commit messages
- Ensure your code passes all tests
- Add tests for new functionality
- Update documentation as needed

## Reporting Issues

If you find a bug or have a feature request, please [open an issue](https://github.com/vercel/ai-chatbot/issues/new) on GitHub. Please include as much detail as possible, including steps to reproduce the issue.

## Pull Request Process

1. Ensure your code follows our style guide
2. Add tests for any new functionality
3. Update documentation if necessary
4. Describe your changes in the pull request description
5. Link any related issues

## Code of Conduct

Please note that this project is released with a Contributor Code of Conduct. By participating in this project you agree to abide by its terms.

## Questions?

If you have any questions about contributing, feel free to reach out by opening an issue or contacting the maintainers.
